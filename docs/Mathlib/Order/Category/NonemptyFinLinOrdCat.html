    Nonempty finite linear orders #

    This defines NonemptyFinLinOrdCat, the category of nonempty finite linear orders with monotone maps. This is the index category for simplicial objects.

    Note: NonemptyFinLinOrd is not a subcategory of FinBddDistLat because its morphisms do not preserve ⊥ and ⊤.
