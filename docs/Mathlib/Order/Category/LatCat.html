    The category of lattices #

    This defines LatCat, the category of lattices.

    Note that LatCat doesn't correspond to the literature definition of [LatCat] (https://ncatlab.org/nlab/show/Lat) as we don't require bottom or top elements. Instead, LatCat corresponds to BddLatCat.
