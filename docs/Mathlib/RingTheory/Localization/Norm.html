    theorem Algebra.norm_localization (R : Type u_1) {S : Type u_2} [inst : CommRing R] [inst : CommRing S] [inst : Algebra R S] {Rₘ : Type u_3} {Sₘ : Type u_4} [inst : CommRing Rₘ] [inst : Algebra R Rₘ] [inst : CommRing Sₘ] [inst : Algebra S Sₘ] (M : Submonoid R) [inst : IsLocalization M Rₘ] [inst : IsLocalization (Algebra.algebraMapSubmonoid S M) Sₘ] [inst : Algebra Rₘ Sₘ] [inst : Algebra R Sₘ] [inst : IsScalarTower R Rₘ Sₘ] [inst : IsScalarTower R S Sₘ] [inst : Module.Free R S] [inst : Module.Finite R S] (a : S) :
    ↑(Algebra.norm Rₘ) (↑(algebraMap S Sₘ) a) = ↑(algebraMap R Rₘ) (↑(Algebra.norm R) a)

    Let S be an extension of R and Rₘ Sₘ be localizations at M of R S respectively. Then the norm of a : Sₘ over Rₘ is the norm of a : S over R if S is free as R-module.
