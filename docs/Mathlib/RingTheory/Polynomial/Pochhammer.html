    The Pochhammer polynomials #

    We define and prove some basic relations about pochhammer S n : S[X] := X * (X + 1) * ... * (X + n - 1) which is also known as the rising factorial. A version of this definition that is focused on Nat can be found in Data.Nat.Factorial as Nat.ascFactorial.

    Implementation #

    As with many other families of polynomials, even though the coefficients are always in ℕ, we define the polynomial with coefficients in any [Semiring S].

    noncomputable def pochhammer (S : Type u) [inst : Semiring S] :
    ℕ → Polynomial S

    pochhammer S n is the polynomial X * (X+1) * ... * (X + n - 1), with coefficients in the semiring S.

    Equations
    • pochhammer S 0 = 1
    • pochhammer S (Nat.succ n) = Polynomial.X * Polynomial.comp (pochhammer S n) (Polynomial.X + 1)
