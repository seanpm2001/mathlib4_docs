    Isomorphisms of R-algebras #

    This file defines bundled isomorphisms of R-algebras.

    Main definitions #

    • AlgEquiv R A B: the type of R-algebra isomorphisms between A and B.

    Notations #

    • A ≃ₐ[R] B : R-algebra equivalence from A to B.

    Type (maxvw)

    • The proposition that the function preserves multiplication

      map_mul' : ∀ (x y : A), Equiv.toFun toEquiv (x * y) = Equiv.toFun toEquiv x * Equiv.toFun toEquiv y

    • The proposition that the function preserves addition

      map_add' : ∀ (x y : A), Equiv.toFun toEquiv (x + y) = Equiv.toFun toEquiv x + Equiv.toFun toEquiv y

    • An equivalence of algebras commutes with the action of scalars.

      commutes' : ∀ (r : R), Equiv.toFun toEquiv (↑(algebraMap R A) r) = ↑(algebraMap R B) r

    An equivalence of algebras is an equivalence of rings commuting with the actions of scalars.
