    Bicategories #

    In this file we define typeclass for bicategories.

    A bicategory B consists of

    • objects a : B,
    • 1-morphisms f : a ‚ü∂ b between objects a b : B, and
    • 2-morphisms Œ∑ : f ‚ü∂ g between 1-morphisms f g : a ‚ü∂ b between objects a b : B.

    We use u, v, and w as the universe variables for objects, 1-morphisms, and 2-morphisms, respectively.

    A typeclass for bicategories extends CategoryTheory.CategoryStruct typeclass. This means that we have

    • a composition f ‚â´ g : a ‚ü∂ c for each 1-morphisms f : a ‚ü∂ b and g : b ‚ü∂ c, and
    • an identity ùüô a : a ‚ü∂ a for each object a : B.

    For each object a b : B, the collection of 1-morphisms a ‚ü∂ b has a category structure. The 2-morphisms in the bicategory are implemented as the morphisms in this family of categories.

    The composition of 1-morphisms is in fact an object part of a functor (a ‚ü∂ b) ‚•§ (b ‚ü∂ c) ‚•§ (a ‚ü∂ c). The definition of bicategories in this file does not require this functor directly. Instead, it requires the whiskering functions. For a 1-morphism f : a ‚ü∂ b and a 2-morphism Œ∑ : g ‚ü∂ h between 1-morphisms g h : b ‚ü∂ c, there is a 2-morphism whiskerLeft f Œ∑ : f ‚â´ g ‚ü∂ f ‚â´ h. Similarly, for a 2-morphism Œ∑ : f ‚ü∂ g between 1-morphisms f g : a ‚ü∂ b and a 1-morphism f : b ‚ü∂ c, there is a 2-morphism whiskerRight Œ∑ h : f ‚â´ h ‚ü∂ g ‚â´ h. These satisfy the exchange law whiskerLeft f Œ∏ ‚â´ whiskerRight Œ∑ i = whiskerRight Œ∑ h ‚â´ whiskerLeft g Œ∏, which is required as an axiom in the definition here.

                Simp-normal form for 2-morphisms #

                Rewriting involving associators and unitors could be very complicated.
