    Linear Functors #

    An additive functor between two R-linear categories is called linear if the induced map on hom types is a morphism of R-modules.

    Implementation details #

    Functor.Linear is a Prop-valued class, defined by saying that for every two objects X and Y, the map F.map : (X ‚ü∂ Y) ‚Üí (F.obj X ‚ü∂ F.obj Y) is a morphism of R-modules.
