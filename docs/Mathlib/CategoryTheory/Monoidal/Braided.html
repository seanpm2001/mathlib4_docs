    Braided and symmetric monoidal categories #

    The basic definitions of braided monoidal categories, and symmetric monoidal categories, as well as braided functors.

    Implementation note #

    We make BraidedMonoidalCategory another typeclass, but then have SymmetricMonoidalCategory extend this. The rationale is that we are not carrying any additional data, just requiring a property.
