    Essentially small categories. #

    A category given by (C : Type u) [Category.{v} C] is w-essentially small if there exists a SmallModel C : Type w equipped with [SmallCategory (SmallModel C)].

    A category is w-locally small if every hom type is w-small.

    The main theorem here is that a category is w-essentially small iff the type Skeleton C is w-small, and C is w-locally small.

                        CategoryTheory.EssentiallySmall C ↔ Small (CategoryTheory.Skeleton C) ∧ CategoryTheory.LocallySmall C
