    Bounded continuous functions #

    The type of bounded continuous functions taking values in a metric space, with the uniform distance.

    source
    structure BoundedContinuousFunction (α : Type u) (β : Type v) [inst : TopologicalSpace α] [inst : PseudoMetricSpace β] extends ContinuousMap :
    Type (maxuv)
    • map_bounded' : ∃ C, ∀ (x y : α), dist (ContinuousMap.toFun toContinuousMap x) (ContinuousMap.toFun toContinuousMap y) ≤ C

    α →ᵇ β is the type of bounded continuous functions α → β from a topological space to a metric space.

    When possible, instead of parametrizing results over (f : α →ᵇ β), you should parametrize over (F : Type _) [BoundedContinuousMapClass F α β] (f : F).

    When you extend this structure, make sure to extend BoundedContinuousMapClass.
