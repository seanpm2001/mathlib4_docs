    Strong topologies on the space of continuous linear maps #

    In this file, we define the strong topologies on E ‚ÜíL[ùïú] F associated with a family ùîñ : Set (Set E) to be the topology of uniform convergence on the elements of ùîñ (also called the topology of ùîñ-convergence).

    The lemma UniformOnFun.continuousSMul_of_image_bounded tells us that this is a vector space topology if the continuous linear image of any element of ùîñ is bounded (in the sense of Bornology.IsVonNBounded).

    We then declare an instance for the case where ùîñ is exactly the set of all bounded subsets of E, giving us the so-called "topology of uniform convergence on bounded sets" (or "topology of bounded convergence"), which coincides with the operator norm topology in the case of NormedSpaces.

    Other useful examples include the weak-* topology (when ùîñ is the set of finite sets or the set of singletons) and the topology of compact convergence (when ùîñ is the set of relatively compact sets).

    Main definitions #

    • ContinuousLinearMap.strongTopology is the topology mentioned above for an arbitrary ùîñ.
    • ContinuousLinearMap.topologicalSpace is the topology of bounded convergence. This is declared as an instance.

    Main statements #

    • ContinuousLinearMap.strongTopology.topologicalAddGroup and ContinuousLinearMap.strongTopology.continuousSMul show that the strong topology makes E ‚ÜíL[ùïú] F a topological vector space, with the assumptions on ùîñ mentioned above.
    • ContinuousLinearMap.topologicalAddGroup and ContinuousLinearMap.continuousSMul register these facts as instances for the special case of bounded convergence.
