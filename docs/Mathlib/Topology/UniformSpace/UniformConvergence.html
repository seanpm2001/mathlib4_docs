    Uniform convergence #

    A sequence of functions F‚Çô (with values in a metric space) converges uniformly on a set s to a function f if, for all Œµ > 0, for all large enough n, one has for all y ‚àà s the inequality dist (f y, F‚Çô y) < Œµ. Under uniform convergence, many properties of the F‚Çô pass to the limit, most notably continuity. We prove this in the file, defining the notion of uniform convergence in the more general setting of uniform spaces, and with respect to an arbitrary indexing set endowed with a filter (instead of just ‚Ñï with atTop).

    Main results #

    Let Œ± be a topological space, Œ≤ a uniform space, F‚Çô and f be functions from Œ± to Œ≤ (where the index n belongs to an indexing type Œπ endowed with a filter p).

    • TendstoUniformlyOn F f p s: the fact that F‚Çô converges uniformly to f on s. This means that, for any entourage u of the diagonal, for large enough n (with respect to p), one has (f y, F‚Çô y) ‚àà u for all y ‚àà s.
    • TendstoUniformly F f p: same notion with s = univ.
    • TendstoUniformlyOn.continuousOn: a uniform limit on a set of functions which are continuous on this set is itself continuous on this set.
    • TendstoUniformly.continuous: a uniform limit of continuous functions is continuous.
    • TendstoUniformlyOn.tendsto_comp: If F‚Çô tends uniformly to f on a set s, and g‚Çô tends to x within s, then F‚Çô g‚Çô tends to f x if f is continuous at x within s.
    • TendstoUniformly.tendsto_comp: If F‚Çô tends uniformly to f, and g‚Çô tends to x, then F‚Çô g‚Çô tends to f x.

    We also define notions where the convergence is locally uniform, called TendstoLocallyUniformlyOn F f p s and TendstoLocallyUniformly F f p. The previous theorems all have corresponding versions under locally uniform convergence.

    Finally, we introduce the notion of a uniform Cauchy sequence, which is to uniform convergence what a Cauchy sequence is to the usual notion of convergence.

    Implementation notes #

    We derive most of our initial results from an auxiliary definition TendstoUniformlyOnFilter. This definition in and of itself can sometimes be useful, e.g., when studying the local behavior of the F‚Çô near a point, which would typically look like TendstoUniformlyOnFilter F f p (ùìù x). Still, while this may be the "correct" definition (see tendstoUniformlyOn_iff_tendstoUniformlyOnFilter), it is somewhat unwieldy to work with in practice. Thus, we provide the more traditional definition in TendstoUniformlyOn.

    Most results hold under weaker assumptions of locally uniform approximation. In a first section, we prove the results under these weaker assumptions. Then, we derive the results on uniform convergence from them.
