    Sums of binomial coefficients #

    This file includes variants of the binomial theorem and other results on sums of binomial coefficients. Theorems whose proofs depend on such sums may also go in this file for import reasons.

    theorem Commute.add_pow {R : Type u_1} [inst : Semiring R] {x : R} {y : R} (h : Commute x y) (n : ℕ) :
    (x + y) ^ n = Finset.sum (Finset.range (n + 1)) fun m => x ^ m * y ^ (n - m) * ↑(Nat.choose n m)

    (x + y) ^ n = Finset.sum (Finset.Nat.antidiagonal n) fun m => Nat.choose n m.fst • (x ^ m.fst * y ^ m.snd)

    theorem Finset.sum_choose_succ_mul {R : Type u_1} [inst : NonAssocSemiring R] (f : ℕ → ℕ → R) (n : ℕ) :
    (Finset.sum (Finset.range (n + 2)) fun i => ↑(Nat.choose (n + 1) i) * f i (n + 1 - i)) = (Finset.sum (Finset.range (n + 1)) fun i => ↑(Nat.choose n i) * f i (n + 1 - i)) + Finset.sum (Finset.range (n + 1)) fun i => ↑(Nat.choose n i) * f (i + 1) (n - i)

    The sum of (n+1).choose i * f i (n+1-i) can be split into two sums at rank n, respectively of n.choose i * f i (n+1-i) and n.choose i * f (i+1) (n-i).
