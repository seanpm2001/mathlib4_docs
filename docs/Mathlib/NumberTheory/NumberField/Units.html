    Units of a number field #

    We prove results about the group (𝓞 K)ˣ of units of the ring of integers 𝓞 K of a number field K.

    Main results #

    • isUnit_iff_norm: an algebraic integer x : 𝓞 K is a unit if and only if |norm ℚ x| = 1
