    Shadows #

    This file defines shadows of a set family. The shadow of a set family is the set family of sets we get by removing any element from any set of the original family. If one pictures Finset Î± as a big hypercube (each dimension being membership of a given element), then taking the shadow corresponds to projecting each finset down once in all available directions.

    Main definitions #

    • Finset.shadow: The shadow of a set family. Everything we can get by removing a new element from some set.
    • Finset.up_shadow: The upper shadow of a set family. Everything we can get by adding an element to some set.
