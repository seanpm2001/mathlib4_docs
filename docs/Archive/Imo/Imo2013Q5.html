    IMO 2013 Q5 #

    Let ℚ>₀ be the positive rational numbers. Let f : ℚ>₀ → ℝ be a function satisfying the conditions

    1. f(x) * f(y) ≥ f(x * y)
    2. f(x + y) ≥ f(x) + f(y)

    for all x, y ∈ ℚ>₀. Given that f(a) = a for some rational a > 1, prove that f(x) = x for all x ∈ ℚ>₀.
